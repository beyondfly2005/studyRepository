# P3、Redis入门

##### **Redis**

概念：Redis（REmote DIctionary Server）开源的高性能键值对（key-value）数据库

特征：

- 数据间没有比如的关联关系

- 内部采用但新车机制机制工作

- 高性能。

  官方数据 

- 多数据类型支持

  - 字符串
  - 列表类型
  - 散列类型
  - 集合类型
  - 有序集合

##### Redis的应用

- 为热点数据加速查询
- 任务队列，如描述，抢购，购票排队等
- 及时信息查询
- 时效性信息控制，如验证码、投票控制等
- 分布式数据共享
- 消息队列
- 分布式锁
